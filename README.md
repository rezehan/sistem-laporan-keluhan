# Template Laporan Keluhan

Repositori ini berisi kode sumber untuk aplikasi **Template Laporan Keluhan**, sebuah aplikasi web yang dibangun menggunakan Vue 3 dan Vite.

## Fitur Utama

Meskipun masih dalam pengembangan, fitur utama yang direncanakan untuk aplikasi ini adalah:

* **Pengajuan Keluhan:** Pengguna dapat mengirimkan laporan keluhan baru.
* **Manajemen Keluhan:** Admin atau pengguna dapat melihat daftar keluhan yang telah masuk.
* **Pelacakan Status:** Memantau status dari setiap keluhan (misalnya: Terkirim, Diproses, Selesai).
* **Antarmuka Responsif:** Dibangun dengan Vue 3 untuk memastikan pengalaman pengguna yang baik di berbagai perangkat.

## Teknologi yang Digunakan

Proyek ini dibangun menggunakan tumpukan teknologi modern frontend:

* **[Vue 3](https://vuejs.org/):** Framework JavaScript progresif untuk membangun antarmuka pengguna.
* **[Vite](https://vitejs.dev/):** Alat build frontend yang sangat cepat.
* **[Vue Router](https://router.vuejs.org/):** Library routing resmi untuk Vue.js.
* **JavaScript (ES6+):** Bahasa pemrograman utama.
* **HTML5 & CSS3:** Struktur dan styling aplikasi.

## Persyaratan Instalasi

Pastikan Anda memiliki perangkat lunak berikut yang terinstal di Template Anda sebelum memulai:

* [Node.js](https://nodejs.org/en/) (disarankan versi LTS)
* [NPM](https://www.npmjs.com/) (biasanya terinstal bersama Node.js)

## Instalasi dan Penggunaan

Ikuti langkah-langkah berikut untuk menjalankan proyek ini secara lokal:

1.  **ekstrak file ini**

2.  **Instal dependensi proyek:**
    Gunakan npm untuk menginstal semua paket yang diperlukan.
    ```bash
    npm install
    ```

3.  **Jalankan server pengembangan:**
    Perintah ini akan menjalankan aplikasi dalam mode pengembangan dengan *hot-reload*.
    ```bash
    npm run dev
    ```

4.  **Buka di browser:**
    Aplikasi akan tersedia di `http://localhost:5173` (atau port lain yang ditampilkan di terminal Anda).

5.  **Build untuk Produksi:**
    Untuk membuat versi produksi dari aplikasi:
    ```bash
    npm run build
    ```
